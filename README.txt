Ticket Productivity Dashboard

This project was built to solve a real-world need: tracking and visualizing daily ticket resolution performance.

The system allows recording resolved tickets by date and category, generating monthly reports, and exporting structured PDF summaries.

- Tech Stack:

PHP (vanilla)

MySQL

PDO

Bootstrap (responsive layout)

DomPDF (PDF export)

- Features: 

Ticket CRUD (Create, Read, Update, Delete)

Monthly filtering

Aggregated SQL queries (GROUP BY date and category)

Pagination

Responsive layout (mobile-ready)

Dynamic PDF report generation

Clean UI with feedback toasts and modal interactions
